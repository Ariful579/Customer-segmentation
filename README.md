# Customer-segmentation

** Executive Summary: Marketing Campaign Customer Segmentation **
Project Overview
This project focuses on Unsupervised Machine Learning and Data Engineering to perform advanced customer segmentation on a dataset of 22,160 customer interactions. By engineering demographic, socioeconomic, and purchasing behaviors, the project leverages dimensionality reduction and hierarchical/iterative clustering to discover hidden consumer archetypes. The ultimate objective is to transform raw interaction logs into actionable data-driven marketing segments, enabling automated, hyper-targeted promotional campaigns.

** Core Architecture & Engineering PipelineExploratory Data Analysis (EDA) & Data Cleansing:**
Handled missing socioeconomic values (Income) via standard row deletion. Transformed temporal features (Dt_Customer) to calculate long-term customer tenure (Customer_For).  Feature Engineering & Dimensionality Reduction:Aggregated continuous spending metrics across product lines (Wines, Fruits, Meat, Fish, Sweets, Gold) into a global Spent metric.  Synthesized categorical data (Marital Status, Education) into structured behavioral fields (Living_With, Family_Size, Is_Parent, Education grouping).  Applied StandardScaler to normalize scale differences across data structures, followed by Principal Component Analysis (PCA) to reduce multicollinearity and select the most impactful variance vectors.  Clustering Algorithms Applied: Evaluated and implemented K-Means Clustering combined with a KElbowVisualizer to determine optimal segment counts ($K$). Validated using Agglomerative Hierarchical Clustering for robust, bottom-up group validation.  Data Visualization Engine: Utilized matplotlib, seaborn pairplots, and mpl_toolkits.3D to map clusters back to business-critical dimensions like income vs. spending power.
